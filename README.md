# ChatApplication_using_Java

## Overview
ChatApplication_using_Java is a simple real-time chat application built using Java. It enables multiple users to communicate over a network using a client-server architecture.

## Features
- Multi-client support
- Real-time messaging
- User-friendly GUI (if applicable)
- Secure message transmission
- Server-side logging
- User authentication (if implemented)

## Technologies Used
- Java (JDK 8 or later)
- Socket Programming
- Swing (for GUI, if applicable)
- JDBC (for database, if used)

## Installation & Setup
### Prerequisites
- Install [Java JDK](https://www.oracle.com/java/technologies/javase-downloads.html)
- (Optional) Install MySQL if using a database

### Steps to Run
1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/ChatApplication_using_Java.git
   ```
2. Navigate to the project directory:
   ```sh
   cd ChatApplication_using_Java
   ```
3. Compile the Java files:
   ```sh
   javac Server.java Client.java
   ```
4. Run the server:
   ```sh
   java Server
   ```
5. Start the client (multiple instances for multiple users):
   ```sh
   java Client
   ```

## Usage
- Start the server first to listen for incoming connections.
- Run multiple client instances to simulate different users.
- Enter your username and start chatting with connected users.
